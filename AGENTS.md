# LESSONS

- Blogger data tags must use legal XML element names; expression calls such as `date.format(...)` cannot appear inside `<data:.../>` and must be replaced with a supported data field or an `expr:*` attribute.
- Treat this repository's `docs/superpowers/` spec-and-plan workflow as intentional; do not require or flag missing TRIP `docs/ARCHI.md` or `docs/1-plans/` artifacts.
